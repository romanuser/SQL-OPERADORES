# SQL Queries

Repositório contendo consultas SQL desenvolvidas para estudo, prática e resolução de problemas envolvendo bancos de dados relacionais.

## Objetivo

Este projeto reúne exemplos de consultas SQL aplicadas em diferentes cenários, com foco no desenvolvimento de habilidades em manipulação e análise de dados.

Os scripts abordam desde consultas básicas até operações mais avançadas, servindo como material de estudo e portfólio.

## Tecnologias

- SQL
- SQL Server
- Banco de Dados Relacional

## Conteúdo

Entre os exemplos presentes neste repositório estão:

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN
- FULL JOIN
- UNION
- Subqueries
- Funções de agregação
- CASE
- CTE (Common Table Expressions)
- Views
- Manipulação de dados (INSERT, UPDATE e DELETE)

## Como utilizar

1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/nome-do-repositorio.git
```

2. Abra o script no SQL Server Management Studio (SSMS) ou na ferramenta de sua preferência.

3. Execute as consultas utilizando um banco de dados compatível.
